auto ключевое слово языка Си.

auto это устаревшее ключевое слово Си, обозначающее локальную область видимости, и в настоящее время его можно опускать. В С++ данное ключевое слово имеет другое значение. 


auto int a;
auto a; // Аналогично предыдущей записи так как в Си по умолчанию все типы - int.
int a; // Современная форма записи
